🔧 HOW IT WORKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

THROTTLING:
  Instead of running every frame (1000+ times/sec), modules now
  run at sensible rates (30-60 FPS) where human eye can't tell
  the difference.

TABLE POOLING:
  Reuses tables instead of creating new ones every frame.
  Example: KillAura was creating 100+ tables/sec = GC pressure.

CACHING:
  Expensive operations (entity lookups, target scans) are cached
  for 33ms. If result is fresh, use cached value instead of
  recalculating.

AUTO GC:
  Every 30 seconds, cleans up old caches and runs garbage
  collection if memory > 50MB.
